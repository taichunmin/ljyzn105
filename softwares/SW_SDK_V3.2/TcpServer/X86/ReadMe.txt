Warning: Using TcpServer Mode, you need to using ReaderSoft to set Reader's Interface in TCP/WIFI/GPRS/4G mode, and WorkMode in ActiveMode, and Set DistinIP and DistinPort in NetParam (DistinPort is ListeningPort).  The Max TcpClient number support 256 .

提示：如果使用TcpServer模式，你需要使用读卡器软件设置读卡器的上传接口是TCP/WIFI/GPRS/4G，同时设置工作模式为 主动模式，同时在网络设置中设置 远程IP和远程端口（远程端口对应TcpServer下的ListeninigPort）。最大支持256个客户端同时连接。
