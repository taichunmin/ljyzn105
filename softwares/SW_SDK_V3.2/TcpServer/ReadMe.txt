Warning: Using TcpServer Mode, you need to using ReaderSoft to set Reader's Interface in RJ45/WIFI/GPRS/4G mode, and WorkMode in ActiveMode, and Set DistinIP and DistinPort in NetParam (DistinPort is ListeningPort).  The Max TcpClient number support 256 .
If you want  execute C++ Version,  you need setup Microsoft Visual C++ 2008 Redistributable X86 or X64.

The reader support customed protocol, such as HTTP, MQTT, LLRP. If you need, please contact your saler.

[Chinese]
提示：如果使用TcpServer模式，你需要使用读卡器软件设置读卡器的上传接口是RJ45/WIFI/GPRS/4G，同时设置工作模式为 主动模式，同时在网络设置中设置 远程IP和远程端口（远程端口对应TcpServer下的ListeninigPort）。最大支持256个客户端同时连接。
如果要打开C++版本的程序，请先安装Microsoft Visual C++ 2008 Redistributable X86 或者 X64 版本.
读卡器支持定制协议，例如HTTP, MQTT, LLRP。如果需要请联系销售客服。
