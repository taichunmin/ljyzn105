Warning: Using TcpServer Mode, you need to using ReaderSoft to set Reader's Interface in RJ45/WIFI/GPRS/4G mode, and WorkMode in ActiveMode, and Set DistinIP and DistinPort in NetParam (DistinPort is ListeningPort).  

The reader support customed protocol, such as HTTP, MQTT, LLRP. If you need, please contact your saler.
