[English]
Warning: Only Support RS232/RS485, if you want to read tag actively, you must use ReaderSoft to set interface to RS232/RS485, and set WorkMode in ActiveMode. 

Compile with Eclipse, and java version is 1.8, and you should install windowbuilder in the eclipse

please copy libSWComApi.so to system lib first
sudo  cp ~/libSWComApi.so /lib/libSWComApi.so
